# 木林森图片上传系统用户手册

## 📖 系统简介

木林森图片上传系统是一个专业的批量图片上传工具，支持Excel表格导入商品信息，智能匹配图片文件，并批量上传到服务器。

### ✨ 主要功能
- 🔐 用户身份认证
- 📊 Excel表格解析（支持.xlsx和.xls格式）
- 🖼️ 图片智能匹配（支持多种图片格式）
- 📁 批量文件夹选择
- ⚡ 高效并发上传
- 📋 实时进度监控
- 📝 详细操作日志

## 📝 操作步骤

### 第一步：用户登录

1. **启动应用程序**
   - 双击桌面上的"木林森图片上传系统"图标
   - 或运行 `upload-client.exe`

2. **输入登录信息**
   - 用户名：输入您的用户名
   - 密码：输入您的密码
   - 点击"登录"按钮

3. **登录验证**
   - 系统会验证您的身份信息
   - 登录成功后，用户名显示为绿色
   - 登录失败会显示错误提示

### 第二步：选择图片类型

1. **图片类型选择**
   - 在下拉菜单中选择图片类型
   - 可选类型：主图、细节图、包装图、使用图等
   - 选择后点击"确认类型"

### 第三步：上传Excel表格

1. **准备Excel文件**
   - 确保Excel文件包含商品编号列
   - 支持格式：.xlsx 和 .xls
   - 建议文件大小不超过10MB

2. **上传表格**
   - 点击"选择Excel文件"按钮
   - 浏览并选择您的Excel文件
   - 系统会自动解析表格内容

3. **验证解析结果**
   - 查看右侧日志区域的解析结果
   - 确认商品编号正确识别
   - 如有问题，请检查Excel格式

### 第四步：选择图片文件夹

1. **选择图片目录**
   - 点击"选择图片文件夹"按钮
   - 浏览并选择包含图片的文件夹
   - 系统支持子文件夹扫描

2. **图片格式支持**
   - 支持格式：.jpg, .jpeg, .png, .bmp, .gif
   - 文件名建议包含商品编号便于匹配
   - 单个图片文件大小建议不超过5MB

3. **智能匹配**
   - 系统会自动匹配商品编号和图片文件名
   - 匹配规则：文件名包含商品编号
   - 匹配结果会在日志区域显示

### 第五步：开始上传

1. **检查匹配结果**
   - 确认所有需要上传的图片都已正确匹配
   - 检查上传状态列，确保没有错误

2. **开始批量上传**
   - 点击"开始上传"按钮
   - 系统开始并发上传图片
   - 进度条显示整体上传进度

3. **监控上传过程**
   - 实时查看每个文件的上传状态
   - 绿色表示成功，红色表示失败
   - 可以随时暂停或取消上传

## 📊 状态说明

### 比对状态
- **✅ 匹配**：商品编号与图片文件名成功匹配
- **❌ 不匹配**：未找到对应的图片文件
- **⚠️ 多匹配**：找到多个匹配的图片文件

### 上传状态
- **⏳ 等待中**：文件在上传队列中等待
- **📤 上传中**：文件正在上传
- **✅ 成功**：文件上传完成
- **❌ 失败**：文件上传失败
- **⏸️ 暂停**：上传已暂停

## ⚙️ 高级功能

### 批量操作
- **全选/取消全选**：快速选择所有文件
- **按状态筛选**：只显示特定状态的文件
- **重新匹配**：重新执行图片匹配逻辑

### 日志管理
- **导出日志**：将操作日志导出为CSV文件
- **清空日志**：清除当前会话的日志记录
- **搜索过滤**：根据关键词过滤日志内容

### 设置选项
- **上传并发数**：调整同时上传的文件数量
- **重试次数**：设置上传失败时的重试次数
- **超时时间**：设置网络请求的超时时间

## ❗ 注意事项

### 文件要求
1. **Excel文件**
   - 必须包含商品编号列
   - 编码格式建议使用UTF-8
   - 避免使用特殊字符

2. **图片文件**
   - 文件名建议包含商品编号
   - 避免使用中文路径
   - 确保文件没有被其他程序占用

### 网络要求
1. **网络连接**
   - 确保客户端能访问服务器
   - 建议使用稳定的网络连接
   - 大批量上传时注意网络带宽

2. **防火墙设置**
   - 确保防火墙允许程序访问网络
   - 如有企业防火墙，请联系IT部门

### 性能建议
1. **批量大小**
   - 建议单次上传不超过1000个文件
   - 大文件建议分批上传
   - 避免在网络高峰期上传

2. **系统资源**
   - 确保有足够的内存空间
   - 关闭不必要的其他程序
   - 定期清理临时文件

## 🔧 故障排除

### 常见问题

1. **登录失败**
   - 检查用户名和密码是否正确
   - 确认服务器连接正常
   - 联系管理员重置密码

2. **Excel解析失败**
   - 检查文件格式是否正确
   - 确认文件没有被其他程序打开
   - 尝试另存为新的Excel文件

3. **图片匹配失败**
   - 检查文件名是否包含商品编号
   - 确认图片格式是否支持
   - 检查文件路径是否正确

4. **上传失败**
   - 检查网络连接状态
   - 确认服务器运行正常
   - 检查文件是否损坏

### 错误代码
- **E001**：网络连接超时
- **E002**：文件格式不支持
- **E003**：文件大小超限
- **E004**：服务器内部错误
- **E005**：权限不足
