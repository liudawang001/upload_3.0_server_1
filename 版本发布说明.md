# 木林森图片上传系统 v1.0.0 发布说明

- **版本号**：v1.0.0
- **发布日期**：2025年9月11日

- **技术栈**：JDK8 + JavaFX8 + Spring Boot 2.1.18 + MySQL 8.0

## 📋 系统概述

木林森图片上传系统是一个专业的企业级批量图片上传解决方案，采用客户端-服务端分离架构，支持Excel表格导入、智能图片匹配和高效并发上传。

### 🏗️ 核心架构
- **客户端**：JavaFX 8 桌面应用程序（Windows）
- **服务端**：Spring Boot 2.1.18 REST API服务
- **数据库**：MySQL 8.0 数据存储
- **通信协议**：HTTP + JSON + Base64图片传输
