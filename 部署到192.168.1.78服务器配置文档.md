# 木林森图片上传系统部署到192.168.1.78服务器配置文档

## 📋 项目概述

本文档详细说明将木林森图片上传系统部署到192.168.1.78服务器所需的所有配置修改。

### 🏗️ 系统架构
- **客户端**：JavaFX 8 桌面应用程序
- **服务端**：Spring Boot 2.1.18 REST API服务
- **数据库**：MySQL 8.0 (47.110.46.27:3306)
- **目标服务器**：192.168.1.78

## 🚀 部署步骤

### 第一步：修改配置文件

1. **修改客户端配置**
   ```bash
   # 编辑客户端主配置文件
   notepad upload-client/src/main/resources/application.properties
   notepad upload-client/src/main/resources/config.properties
   
   # 编辑客户端工具类
   notepad upload-client/src/main/java/com/mls/upload/client/util/ConfigUtil.java
   
   # 编辑客户端启动脚本
   notepad upload-client/start-client.bat
   ```

2. **修改服务端配置**
   ```bash
   # 编辑服务端环境配置文件
   notepad upload-server/src/main/resources/application-dev.yml
   notepad upload-server/src/main/resources/application-prod.yml
   
   # 编辑服务端配置类（如需要）
   notepad upload-server/src/main/java/com/mls/upload/server/config/CacheRefreshProperties.java
   ```

### 第二步：重新编译打包

1. **编译客户端**
   ```bash
   cd upload-client
   mvn clean package -DskipTests
   ```

2. **编译服务端**
   ```bash
   cd upload-server
   mvn clean package -DskipTests
   ```

### 第三步：部署到目标服务器

1. **上传服务端文件到192.168.1.78**
   ```bash
   # 上传服务端jar包和配置文件
   scp upload-server/target/upload-server.jar user@192.168.1.78:/opt/mls-upload/
   scp upload-server/start-server.bat user@192.168.1.78:/opt/mls-upload/
   ```

2. **在192.168.1.78服务器上启动服务**
   ```bash
   # 在目标服务器上
   cd /opt/mls-upload/
   java -jar upload-server.jar
   ```

3. **验证服务端启动**
   ```bash
   # 检查服务是否正常启动
   curl http://192.168.1.78:8081/api/auth/health
   ```

### 第四步：部署客户端

1. **分发客户端到用户机器**
   - 将编译后的 `upload-client/target/upload-client.exe` 分发给用户
   - 或者分发整个 `upload-client/target/` 目录（包含jar和lib文件夹）

2. **客户端连接测试**
   - 运行客户端应用
   - 测试登录功能
   - 测试文件上传功能
