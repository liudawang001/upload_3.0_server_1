# 图片上传系统开发任务书

## 项目基本信息

**项目名称**: 木林森图片信息上传客户端系统  
**开发周期**: 15个工作日  
**技术要求**: JDK8、JavaFX、Spring Boot、MySQL  
**目标平台**: Windows 7及以上系统  
**项目类型**: 客户端-服务端分离架构  

## 项目概述

开发一个基于JavaFX的Windows桌面客户端应用程序，用于批量上传图片到远程服务器，并管理相关的元数据信息。系统支持用户认证、Excel表格解析、图片分类上传、进度监控和日志管理等功能。

## 技术架构

### 客户端技术栈
- **JDK版本**: OpenJDK 8
- **UI框架**: JavaFX 8 (JDK8内置)
- **构建工具**: Maven 3.6+
- **HTTP客户端**: OkHttp 3.14.9
- **JSON处理**: Jackson 2.10.5
- **Excel处理**: Apache POI 4.1.2
- **日志框架**: Logback 1.2.3
- **打包工具**: launch4j

### 服务端技术栈
- **框架**: Spring Boot 2.1.18.RELEASE
- **数据访问**: MyBatis 3.5.6
- **数据库**: MySQL 8.0
- **密码加密**: BCrypt
- **连接池**: HikariCP
